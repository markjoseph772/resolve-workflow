
# resolve-workflow
> Control DaVinci Resolve with plain English via OpenClaw / Claude Code.

Import footage, organize bins, create timelines, apply LUTs, and render — all from a single chat command. No more clicking through menus.

## What it does
- Batch import footage into organized bins (`--dry-run` to preview first)
- Create & configure timelines (resolution, fps, based on footage)
- Apply LUTs to every clip on a timeline
- Render and export (H.264, ProRes, DNxHD, etc.)

## Platform support
- **Windows 10/11** · **macOS 12+**
- DaVinci Resolve 17, 18, 19 (Free & Studio)
